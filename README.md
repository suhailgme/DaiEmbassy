## Dai Embassy https://www.daiembassy.online/

### Local Front-End Usage*
##### *Relies on Docker hosted API for application data (Dokku)
Clone Repo: 

``` git clone https://github.com/suhailgme/DaiEmbassy.git ```

Get Dependencies:

``` npm install ```

Run Test Server:

``` npm start ```

Create Build:

``` npm run build ```
